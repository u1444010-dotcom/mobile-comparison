📱 TechCompare Hub

A modern smart phone comparison website with AI recommendations, visual graphs, and product cards. Built using HTML, CSS, and JavaScript.

🚀 Features
🔍 Smart Search
Search phones in real-time
Instant suggestions dropdown
Auto-select comparison phones
⚖️ Phone Comparison Tool
Compare two phones side-by-side
RAM, Camera, Battery comparison
Buy Now affiliate links support
🏆 Auto Winner System
Automatically highlights the better phone
Based on specs scoring system
📊 Visual Graphs
Animated comparison bars
RAM, Camera, Battery visualization
💰 Best Deal System
Highlights phones under budget
“BEST DEAL” badge automatically added
🧠 AI Phone Recommender
Suggests best phone based on:
Budget 💰
Priority (Camera / Battery / Performance)
Smart filtering system
🎴 Product Cards UI
Beautiful product grid
Hover animations
Buy buttons for each phone
🛠️ Tech Stack
HTML5
CSS3 (Modern UI)
